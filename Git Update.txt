1)

Clone the Repository:

git clone <repository_url>


2)

Navigate to the Repository:

cd <repository_directory>

3)

Create a New Branch (Optional):

git checkout -b <branch_name>

Or,

If it is in main branch, then =>

git checkout main



---------------------------------


4)

Stage Changes:

git add .


5)

Commit Changes:

git commit -m "Updated Commit 1"


6)

git push origin <Include branchname and remove this angle bracket>


7)

Switch to the Main Branch (Optional):

git checkout main


8)

Pull the Latest Changes (Optional):

git pull origin main
